\section{Chapter 6: Advanced Topics in Computability Theory}

\subsection*{6.3}

Show that if A $\le$T B and B $\le$T C, then A $\le$T C.


\subsection*{6.4}

Let $A_{TM}^{ATM} = \{\langle M, w\rangle \mid M$ is an oracle TM and $M^{ATM}$ accepts $w\}$. Show that $A_{TM}^{ATM}$ is undecidable relative to $ATM$.


\subsection*{6.7}

Show that for any two languages A and B, a language J exists, where A $\le$T J and B $\le$T J.


\subsection*{6.10}

Let A and B be two disjoint languages. Say that language C separates A and B if A $\subseteq$ C and B $\subseteq$ C. Describe two disjoint Turing-recognizable languages that aren't separable by any decidable language.


\subsection*{6.12}

In Corollary 4.18, we showed that the set of all languages is uncountable. Use this result to prove that languages exist that are not recognizable by an oracle Turing machine with an oracle for ATM .


\subsection*{6.14}

Show how to compute the descriptive complexity of strings K(x) with an oracle for ATM .


\subsection*{6.17}

Show that the set of incompressible strings is undecidable.


\subsection*{6.22}

Describe two different Turing machines, M and N , where M outputs hN i and N outputs hM i, when started on any input.
